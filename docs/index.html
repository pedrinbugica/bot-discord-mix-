Como funciona o mix

Do momento em que o painel é criado até o registro do resultado, o bot guia tudo automaticamente.

1

Criação do painel

Um admin usa /mix panel e o bot publica uma mensagem com os botões Entrar e Sair. Os jogadores clicam para entrar na fila. O painel mostra em tempo real quantas pessoas já estão na fila.

2

Fila cheia — times sorteados

Quando 10 jogadores entram, o bot avisa a todos, sorteia os times automaticamente e publica os dois grupos: 🔵 Time A e 🔴 Time B.

3

Votação de capitão

Cada time vota em quem vai ser o capitão clicando nos botões. O mais votado é escolhido. Em caso de empate, o bot decide por sorteio. O capitão representa o time no veto de mapas.

4

Veto de mapas BO3

Os capitães alternam banindo e escolhendo mapas. A ordem é sempre a mesma:

Time A BAN
Time B BAN
Time A PICK
Time B PICK
Time A BAN
Time B BAN
Sobra DECIDER
5

Mover para voz (opcional)

Se os canais de voz estiverem configurados, o bot move automaticamente cada time para o canal correto assim que o veto termina.

6

Registro do resultado

Após a partida, um dos capitães usa /win para registrar quem ganhou e o placar. O resultado fica salvo no histórico para consulta futura.

💬 Referência

Comandos disponíveis

Todos os comandos são do tipo slash — basta digitar / no Discord para vê-los.

🎮
/mix
Gerencia o painel da fila. Apenas admins.
/mix panelCria e publica o painel da fila no canal atual.
/mix resetLimpa a fila e remove o painel antigo.
🏆
/win
Registra o resultado da série BO3. Só os capitães da partida atual podem usar.
timeTime vencedor: A ou B.
mapas_aQuantidade de mapas ganhos pelo Time A (0 a 2).
mapas_bQuantidade de mapas ganhos pelo Time B (0 a 2).
📜
/historico
Mostra as partidas passadas com placar, capitães e mapas. Pode filtrar por jogador.
jogadorOpcional — filtra as partidas de um jogador específico.
⚙️
/configurar
Configura os canais de voz do mix neste servidor. Cada servidor tem sua própria configuração.
/configurar canaisDefine lobby, Time A e Time B. Apenas admins.
/configurar verMostra a configuração atual de canais de voz.
🏓
/ping
Testa se o bot está online e respondendo.
🚀 Primeiro uso

Configurando no seu servidor

Siga os passos abaixo para colocar o bot funcionando do zero num servidor novo.

Convide o bot para o servidor

Use o link de convite do bot. O administrador do servidor precisa clicar em Autorizar e aceitar as permissões pedidas.

Crie os canais de voz necessários

Você vai precisar de 3 canais de voz: um Lobby (onde os jogadores esperam), um Time A e um Time B. Os nomes podem ser o que quiser.

Configure os canais com /configurar canais

Digite o comando e selecione os três canais de voz. Só precisa fazer isso uma vez por servidor.

Crie o painel da fila com /mix panel

Use este comando no canal de texto onde quer que a fila fique. O bot publica o painel com os botões de Entrar e Sair.

Pronto! Os jogadores já podem entrar na fila

Quando 10 pessoas clicarem em Entrar, o bot inicia o mix automaticamente.

💡

Dica: Para testar com menos jogadores, o administrador pode ajustar a variável MIX_QUEUE_MAX nas configurações do bot (ex.: 2 para testar com apenas 2 pessoas).

Exemplo de uso do /configurar canais

/configurar canais lobby: #lobby-mix time-a: #time-a-voz time-b: #time-b-voz

Exemplo de uso do /win

# Time A ganhou 2x1 /win time:A mapas_a:2 mapas_b:1 # Time B ganhou 2x0 /win time:B mapas_a:0 mapas_b:2
⚠️

Atenção: Só os capitães da partida atual podem usar o /win. Se nenhum capitão registrar o resultado, o histórico fica marcado como pendente.

🔐 Bot no servidor

Permissões necessárias

O bot precisa dessas permissões para funcionar corretamente. Sem elas, algumas funções podem falhar.

Ver canais
Para ler mensagens e interagir nos canais de texto.
Enviar mensagens
Para publicar o painel, avisos e resultados.
Incorporar links (Embeds)
Para exibir o painel da fila com visual formatado.
Gerenciar mensagens
Para apagar mensagens temporárias após o veto.
Conectar (voz)
Para acessar os canais de voz dos times.
Mover membros
Para mover os jogadores para Time A e Time B após o veto.
ℹ️

Para que o bot possa mover membros entre canais de voz, ele precisa ter essa permissão especificamente nos canais de voz do Time A e Time B, não só no servidor como um todo.

🌐 Vários servidores

Usando em múltiplos servidores

O bot funciona em quantos servidores quiser ao mesmo tempo. Cada servidor tem sua própria fila, histórico e configuração de canais — completamente independentes.

✅ Independente por servidor

Fila, times, histórico de partidas e canais de voz são separados para cada servidor.

✅ Configuração própria

Cada admin configura os canais do seu servidor com /configurar canais.

✅ Uma hospedagem só

O bot roda num único lugar e serve todos os servidores simultaneamente.

✅ Nenhuma interferência

O mix do servidor A não afeta nem aparece no servidor B.

Como adicionar em um novo servidor

Compartilhe o link de convite com o admin do novo servidor

O admin clica no link, escolhe o servidor e autoriza. O bot entra automaticamente.

O admin configura os canais de voz

No novo servidor, o admin usa /configurar canais para definir o lobby e os canais dos times.

Cria o painel com /mix panel

E pronto — o bot está funcionando de forma independente no novo servidor.

💡

Os comandos podem levar até 1 hora para aparecer num servidor novo. Isso é normal — o Discord precisa de tempo para distribuir os comandos globalmente.
